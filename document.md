## Notas Técnicas

- Todas as páginas seguem o mesmo padrão de design com fundo preto
- Uso consistente de badges (pequenos elementos de destaque) em cinza
- Botões padrão: "Button CTA" (branco) e "Learn More" (transparente com borda)
- Responsividade implementada para dispositivos móveis
- Animações implementadas com Framer Motion
- Uso de componentes Shadcn UI para elementos de interface
