6. Notas Técnicas

* Todas as páginas seguem padrão visual com fundo preto
* Uso consistente de badges em cinza
* Botões padrão: "Button CTA" (branco), "Learn More" (transparente com borda)
* Responsividade mobile-first com Tailwind CSS
* Animações com Framer Motion
* Componentes Shadcn UI para interface
* Componentização e reutilização de elementos
* Otimização de performance e acessibilidade
